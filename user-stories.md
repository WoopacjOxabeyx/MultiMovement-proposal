User Story 1  
As a trickshotter, I want to pull off risky strategies so that I can get rewarded more for my efforts.  
Acceptance Criteria 1: The game rewards you more for harder tricks  
Acceptance Criteria 2: The game gives you  opportunities and the tools you need to attempt trickshots.

User Story 2  
As a speedrunner, I want to go as fast as possible, so that I can get a quick time.

Acceptance Criteria 1: The game’s cutscenes are skippable.  
Acceptance Criteria 2: The game’s doors open quicker and earlier based on your movement speed, so that you can go fast through them.

User Story 3  
As a new player, I want to learn the mechanics easily, so that I can get better at the game.

Acceptance Criteria 1: There is a tutorial to teach you the ropes of the game.  
Acceptance Criteria 2: There are easier modes so that you don't have to be perfect to have fun.
